How to install the engine

Method for Windows:

Step 1: Install all the necessary tools for installation

For installation you will need: git, g++, git bash, GNU/Make

Step 2: Clone the project to your computer

Open git bash and type: cd path/to/folder.Then enter the: git clone https://github.com/quandie/synergy-engine. Wait for the project to be installed

Step 3: Dependency installation

Run update-master-g++.bat, after compilation run update-master.exe and enter the command $dwnlib

After downloading the archive, unzip it to the directory in synergy-engine/

Go back to update-master.exe and type $build, wait for compilation. And that's it, the installation is complete!
